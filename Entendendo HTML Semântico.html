Acessibilidade

Acessibilidade pode ser definida como a possibilidade e condição de alcance, percepção e entendimento para a utilização, em igualdade de oportunidades, com segurança e autonomia, do meio físico, do transporte, da informação e da comunicação, inclusive dos sistemas e tecnologias de informação e comunicação, bem como de outros serviços e instalações. Para as pessoas com deficiência e mobilidade reduzida, a acessibilidade possibilita uma vida independente e com participação plena em todos os seus aspectos; e para todas as pessoas, em diferentes contextos, pode proporcionar maior conforto, facilidade de uso, rapidez, satisfação, segurança e eficiência. (W3C Brasil).

Na Internet, o termo acessibilidade refere-se também a recomendações do W3C, que visam permitir que todos possam ter acesso aos sítios, independente de possuírem alguma deficiência ou não. Essas recomendações passam pelo tamanho e cor da fonte, localização dos espaços clicáveis, facilidade de disponibilização de conteúdo e outras sugestões relativas até aos códigos das páginas (HTML e CSS, entre outros).


Dicas de Atalhos que podem ser emplementados e utilizados em uma página web para acessibilidade:
  • ALT + 1 = Redireciona para o conteúdo da página
  • ALT + 2 = Redireciona para o menu
  • ALT + 3 = Redireciona para o campo de busca
  • ALT + 4 = Redireciona para o rodapé
  • ALT + 5 = Aplica ou retira o alto contraste
  • ALT + 6 = Diminui fonte
  • ALT + 7 = Aumenta fonte
  • ALT + 8 = Reseta Fonte
  • ALT + 9 = Redireciona o usuário para a página inicial
  • ALT + 0 = Redireciona o usuário para a página de acessibilidade

    Tags Section, Aside e Nav

    A tag < section > define uma seção em um documento.

    A tag section é utilizada para marcar as seções de conteúdo de uma página. Com Esse elemento agrupamos de forma lógica nosso conteúdo, separando a informação em áreas diferentes. O principal objetivo é retirar essa responsabilidade das divs. Tendo como principal diferencial a navegação semântica com HTML 5.

    Digamos que na sua página inicial você tenha três chamadas principais: Clientes, Serviços e Notícias. Aí um ótimo lugar para você usar a tag section. Cada uma dessas chamadas podem ser um elemento de seção. A tag pode ser usada em conjunto com o H1, H2, H3, H4, H5 e H6, elementos para indicar a estrutura do documento. Ele pode conter um header e um footer se necessário.

    Exemplo:

    A tag section em resumo são seções com ligação com assunto principal do site

    Já a tag aside são seções que não tem ligação com o asunto principal da página como exemplo: fale com o corretor , fazer anuncios e etc...

    O elemento HTML < aside> representa uma seção de uma página que consiste de conteúdo que é tangencialmente relacionado ao conteúdo do seu entorno, que poderia ser considerado separado do conteúdo. Essas seções são, muitas vezes, representadas como barras laterais. Elas muitas vezes contem explicações laterais, como a definição de um glossário; conteúdo vagamente relacionado, como avisos; a biografia do autor; ou, em aplicações web, informações de perfil ou links de blogs relacionados.

    Note: Notas de uso: Não use o elemento < aside> para texto entre parenteses, pois este tipo de texto é considerado parte do fluxo principal.
