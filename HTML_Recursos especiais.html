Tags code, kbd, pre


A tag kbd serve para informar que o texto é uma tecla de teclado

Exemplo:

Para copiar essa informação presione CTRL + C


---------

A tag code serve para informar que o texto é um código

Exemplo:

Para trocar o fundo de um elemento, utilize o comando background-color


---------

A tag pre serve como no exemplo da tag code porem em formato mono space, já com uma borda.

Exemplo:

Comando completo de alterar cor de fundo:

        
//Neste momento você aplica desta forma:

        
        code {
            background-color: rgb(236, 201, 230);
            color: red;
        }
